рҹ“„1. Domains вҖ” N, W, I, Rв–ҫ
REPLACEMENT SETS
An inequation is only solved once you know which numbers x is allowed to be. N = natural numbers (1, 2, 3, вҖҰ), W = whole numbers (0, 1, 2, вҖҰ), I = integers (вҖҰ, вҲ’2, вҲ’1, 0, 1, 2, вҖҰ), R = real numbers (every point on the number line).
Why the domain matters
The algebra gives a range like x вүӨ 6. If x вҲҲ N, the solution set is the list {1, 2, 3, 4, 5, 6}. If x вҲҲ R, the solution set stays the whole interval and is written in set-builder form: {x : x вүӨ 6, x вҲҲ R}.
рҹ”Ғ2. The One Rule That Changes Everythingв–ҫ
вҡ пёҸ MBA Note: Multiplying or dividing BOTH sides by a negative number flips the inequality sign
EXAMPLE
вҲ’2x вүҘ вҲ’12 вҶ’ divide both sides by вҲ’2, and flip вүҘ to вүӨ вҶ’ x вүӨ 6. Adding or subtracting the same number from both sides never flips the sign вҖ” only multiplying/dividing by a negative does.
Safer alternative
If a negative coefficient is making you nervous, move all the x-terms to the side where they end up positive instead of dividing by a negative вҖ” you'll never need to flip the sign that way.

A вүӨ EXPRESSION вүӨ B
Split into two separate inequations sharing the middle expression, solve each one on its own, then combine the two results onto one range.
