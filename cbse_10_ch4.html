Exemplar Sample Q1 Medium 1 Mark
Q. Which one of the following is not a quadratic equation?
(A) (x + 2)² = 2(x + 3)
(B) x² + 3x = (−1)(1 − 3x)²
(C) (x + 2)(x − 1) = x² − 2x − 3
(D) x³ − x² + 2x + 1 = (x + 1)³
WORKING Check option (C):
LHS = (x + 2)(x − 1)
LHS = x² − x + 2x − 2
LHS = x² + x − 2

RHS = x² − 2x − 3

LHS − RHS = (x² + x − 2) − (x² − 2x − 3)
LHS − RHS = x² + x − 2 − x² + 2x + 3
LHS − RHS = 3x + 1

This is a linear expression (highest power of x is 1).
∴ Option (C) is not a quadratic equation.
ANSWER∴ The correct option is (C).

Q. Which of the following is a quadratic equation?

(B) −2x² = (5 − x)2x5

(−1 + 1)x² + 32x = 7
(0)x² + 32x = 7
∴ This is a linear equation, not quadratic.

2x² − 3x + 1 = 0

This is of the form ax² + bx + c = 0.
∴ Option (D) is a quadratic equation.
ANSWER∴ The correct option is (D).

Q. (x² + 1)² − x² = 0 has
(A) four real roots
(B) two real roots
(C) no real roots
(D) one real root
WORKING (x² + 1)² − x² = 0
Expand (x² + 1)² using (a+b)² = a²+2ab+b²:  [using (a+b)² = a²+2ab+b²]
x⁴ + 2x² + 1 − x² = 0
x⁴ + x² + 1 = 0

For any real value of x, x⁴ ≥ 0 and x² ≥ 0.

Q. The equation x + 1x = 3x, (x ≠ 0) is expressed as a quadratic equation in the form ax² + bx + c = 0. The value of a − b + c is:
(A) 5
(B) 2
(C) 1
(D) −1

WORKING x + 1x = 3x

x² + 1 − 3x² = 0
−2x² + 1 = 0

2x² − 1 = 0

∴ a − b + c = 1

GIVEN(x − 2)² + 1 = 2x − 3
FORMULA(a − b)² = a² − 2ab + b²
WORKING LHS = (x − 2)² + 1
LHS = x² − 2(x)(2) + (2)² + 1  [using (a−b)² = a²−2ab+b²]

LHS = x² − 4x + 5

So the equation becomes:
x² − 4x + 5 = 2x − 3
x² − 4x + 5 − 2x + 3 = 0
x² − 6x + 8 = 0

This is of the form ax² + bx + c = 0, where a = 1, b = −6, c = 8.
∴ The given equation is a quadratic equation.
ANSWER∴ (x − 2)² + 1 = 2x − 3 is a quadratic equation, equivalent to x² − 6x + 8 = 0.

Standard Form of a Quadratic Equation ▾
A quadratic equation in the variable x is an equation of the form:

ax² + bx + c = 0, where a, b, c are real numbers and a ≠ 0

Here:
• a = coefficient of x²
• b = coefficient of x
• c = constant term

If a = 0, the equation reduces to bx + c = 0, which is linear, not quadratic.
Roots of a Quadratic Equation ▾
A real number α is called a root of ax² + bx + c = 0 if:

aα² + bα + c = 0

The roots of the quadratic equation ax² + bx + c = 0 are the same as the zeroes of the quadratic polynomial ax² + bx + c.

A quadratic equation can have at most two roots.
Solving by Factorisation ▾
If the quadratic polynomial ax² + bx + c can be factorised into two linear factors, the roots are found by equating each factor to zero.

Method — splitting the middle term:
1. Find a × c
2. Find two numbers whose product is a × c and whose sum is b
3. Split the middle term (bx) into two terms using these numbers
4. Group in pairs and take out the common factor from each pair
5. Take out the common bracket and equate each factor to zero
Quadratic Formula ▾
If b² − 4ac ≥ 0, the real roots of ax² + bx + c = 0 are given by:

x = −b ± √(b² − 4ac)2a

This gives two roots:
x = −b + √(b² − 4ac)2a   and   x = −b − √(b² − 4ac)2a
Discriminant ▾
The expression b² − 4ac is called the discriminant of the quadratic equation ax² + bx + c = 0, usually denoted D.

D = b² − 4ac
Nature of Roots (based on Discriminant) ▾
A quadratic equation ax² + bx + c = 0 has:

• Two distinct real roots, if D > 0
• Two equal real roots (coincident roots), if D = 0
• No real roots, if D < 0

When D = 0, both roots equal −b2a.
Standard Algebraic Identities Used in This Chapter ▾
• (a + b)² = a² + 2ab + b²
• (a − b)² = a² − 2ab + b²
• (a + b)(a − b) = a² − b²
• (a + b)³ = a³ + 3a²b + 3ab² + b³
• (a − b)³ = a³ − 3a²b + 3ab² − b³

• Speed = DistanceTime    (so Time = DistanceSpeed)

• Area of rectangle = Length × Breadth
• Perimeter of rectangle = 2(Length + Breadth)
